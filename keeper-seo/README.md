### 使用说明

### 安装
```javascript
npm install keeper-seo --save-dev
到node_modules/keeper-seo目录下运行
npm link

### 查看当前配置
命令如下：
```javascript
seo //进入cadmin模式
.help //查看所有命令及说明
```
